# Estudio de metodos para la resolucion del TOP
Códigos para el estudio de métodos para la resolución del Team Orienteering Problem

En este trabajo se lleva a cabo un estudio de los métodos de resolución para el Team Orienteering Problem (TOP) aplicado a la optimización del calendario del Campeonato Mundial de Fórmula 1. El objeto de este estudio es maximizar los beneficios económicos totales mediante la selección óptima de carreras dentro de un calendario anual de 52 semanas. Esta investigación consiste en la implementación y comparativa de tres enfoques algorítmicos diferenciados: un método exacto basado en programación lineal entera, un método heurístico constructivo y un método metaheurístico híbrido para instancias de gran escala. Con el objetivo de determinar la eficiencia y rendimiento de cada técnica, se realiza una comparativa exhaustiva entre los tres métodos en términos de calidad de solución y tiempo computacional. Las conclusiones obtenidas demuestran que los métodos propuestos son capaces de encontrar soluciones de calidad competitiva respetando las restricciones temporales, operacionales y comerciales del problema del calendario de F1.
